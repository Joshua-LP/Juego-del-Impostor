## 🤩 ¿Por qué vas a AMAR este juego?

¿Alguna vez has querido un juego que no necesite explicar reglas eternas, que sea perfecto para cualquier reunión y que haga reír a todos desde la primera ronda? **El Impostor** es exactamente eso.


Cada jugador recibe una palabra secreta. **Todos menos uno**. Ese jugador diferente es **el Impostor** 🕵️ — y tiene que sobrevivir sin que nadie lo descubra.

¿El resultado? Debates acalorados, falsas acusaciones, alianzas inesperadas... y carcajadas garantizadas.

---

## 🎮 ¿Cómo se juega?

```
1. 🃏  Cada jugador toca la pantalla y ve su palabra en secreto
2. 🗣️  Por turnos, todos dan una pista sin revelar la palabra directamente  
3. 🤔  El grupo discute: ¿quién está mintiendo? ¿quién da pistas raras?
4. 🗳️  Todos votan al sospechoso... ¿acertarán?
5. 🎭  ¡Se revela quién era el Impostor y cuál era la palabra secreta!
```

Simple de aprender. Imposible de dominar. 😏

## 🚀 Empieza a jugar ahora mismo

No necesitas instalar nada. Solo tienes que abrir el archivo `index.html` en tu navegador:

```bash
# Clona el repositorio
git clone https://github.com/Joshua-LP/Juego-del-Impostor.git

# Entra a la carpeta
cd Juego-del-Impostor

# Abre index.html en tu navegador favorito
# (Doble clic en el archivo o arrástralo al navegador)
```

> 💡 **Tip:** También puedes compartir los archivos directamente con tus amigos y jugarlo sin necesidad de internet.
